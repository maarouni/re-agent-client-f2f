# 🏡 Real Estate Deal Evaluator — Agent & Investor Edition  

A lightweight, Streamlit-powered real estate deal evaluation tool that generates:
- Investor PDF reports  
- Agent-branded client reports  
- Multi-year ROI & cash-flow projections  
- Optional improvements analysis  

This repository contains the full source for the **agent + investor** edition of the tool, including:
- Streamlit app  
- Financial calculation engine  
- PDF generator (investor + agent reports)  
- Optional dual-property comparison
